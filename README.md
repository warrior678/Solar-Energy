# ☀️ Solar Energy Data Analysis using SQL and Power BI

## 📌 Project Overview

This Solar Energy Data Analysis project focuses on analyzing solar power generation data using SQL and Power BI. The dataset was imported from a CSV file into SQL for data cleaning, querying, and aggregation. The processed data was then visualized in Power BI to create an interactive dashboard that provides insights into solar energy production, performance trends, and operational efficiency. This project demonstrates practical data analysis skills in handling energy data and building dashboards to support data-driven decision-making.

---

## 🎯 Project Objectives

* Import solar energy data from a CSV file into SQL
* Perform data cleaning and transformation using SQL
* Analyze solar energy generation and performance trends
* Calculate key performance metrics related to energy production
* Build an interactive dashboard using Power BI

---

## 🛠️ Tools & Technologies Used

* **SQL** – Data querying, filtering, and aggregation
* **Power BI** – Dashboard creation and visualization
* **CSV File** – Data source for analysis

---

## 📊 Key Insights & Analysis

* Total energy generated
* Energy generation trends over time
* Performance comparison across locations or panels
* Peak energy production periods
* Operational efficiency analysis

---

## 📂 Project Workflow

1. Imported solar energy dataset from CSV file into SQL
2. Cleaned and queried data using SQL
3. Loaded processed data into Power BI
4. Created interactive dashboards and visualizations

---

## 📁 Project Files

* **solar_energy_dataset.csv** – Raw dataset
* **solar_energy_queries.sql** – SQL queries used for analysis
* **solar_energy_dashboard.pbix** – Power BI dashboard file

This project demonstrates the ability to analyze renewable energy data using SQL and Power BI. It highlights skills in data querying, data visualization, and dashboard development, which are essential for Data Analyst roles and energy analytics scenarios.
